# AppTalentNavi - サービス仕様書

## 1. サービス概要
- **一言で**: CLIで対話しながらLP（ランディングページ）を作成する研修ツール
- **対象ユーザー**: プログラミング初心者（研修受講者）
- **解決する課題**: コーディング未経験者がAIの力を借りて、簡単にWebページを作れる体験を提供
- **ステータス**: MVP

## 2. ユーザーロールと権限
| ロール | できること |
|---|---|
| 研修受講者 | CLI対話でLP作成、テンプレート利用、HTML確認・修正 |
| 研修講師 | セットアップ支援、カスタムテンプレート追加 |

## 3. 機能一覧
### 実装済み
- [x] CLI対話インターフェース（co-vibeベース）
- [x] Ollama（ローカルLLM）による完全ローカル動作
- [x] LP自動生成（HTML/CSS、単一ファイル）
- [x] ブラウザ自動表示
- [x] /lp コマンド（LP作成ウィザード）
- [x] /open コマンド（HTML再表示）
- [x] 4種類のLPテンプレート
- [x] セッション保存・復元
- [x] 簡易セットアップウィザード

### 未実装（予定）
- [ ] テンプレートプレビュー機能
- [ ] LP書き出し（ZIP/デプロイ）

## 4. 画面一覧
| 画面 | 起動方法 | 概要 |
|---|---|---|
| CLI対話 | `python hajime.py` | メインの対話画面 |
| セットアップ | `python setup-hajime.py` | Ollama確認・モデルDL |
| 生成LP | ブラウザ自動表示 | 生成されたHTML |

## 5. データモデル
### セッション
- JSONL形式で `~/.local/state/co-vibe/sessions/` に保存
- 会話履歴を保持し、`--resume` で復元可能

### テンプレート
- `templates/` ディレクトリに HTML ファイルとして配置
- 単一ファイル（CSS インライン）、外部依存なし

## 6. 外部連携
- LLM: Ollama（ローカル、http://localhost:11434）
- 推奨モデル: qwen2.5-coder:7b

## 7. ビジネスルール
- 完全ローカル動作（インターネット不要、API キー不要）
- 日本語対応必須
- 初心者向けのため、高度な機能は非表示

## 8. 非機能要件
- 想定ユーザー数: 研修1回あたり10-30名
- 動作環境: Windows / macOS / Linux（Python 3.8+）
- Ollama + qwen2.5-coder:7b の最低要件: RAM 8GB

## 9. 既知の課題・制限
- Ollama のモデル品質に依存（7Bモデルでは複雑なLP生成が困難な場合あり）
- ブラウザ自動オープンは環境により動作しない場合がある
