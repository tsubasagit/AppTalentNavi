# templates/ — v1.x レガシーテンプレート

このディレクトリにはAppTalentNavi v1.x で使用していたLP（ランディングページ）テンプレートが含まれています。

v2.0 ではLP作成からビジネスタスク自動化（データ抽出・Web調査・ファイル整理）にピボットしたため、
これらのテンプレートは直接使用されません。

後方互換性のために保持していますが、将来のバージョンで削除される可能性があります。

## 含まれるテンプレート

- `basic-lp.html` — 基本テンプレート
- `cafe-lp.html` — カフェ・飲食店向け
- `service-lp.html` — サービス・SaaS向け
- `portfolio-lp.html` — ポートフォリオ向け
