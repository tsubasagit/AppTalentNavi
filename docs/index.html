動作環境

2つのモードから選べます

クラウドモード おすすめ

  • Windows 10 / 11
  • インターネット接続
  • Gemini APIキー（無料で取得可）
  • Pythonは不要（exe版）

オフラインモード 社内研修向け

  • Windows 10 / 11
  • RAM 8GB以上（16GB推奨）
  • Ollama（自動インストール対応）
  • ネット接続不要（初回以降）

よくある質問

プログラミングの知識は必要？
全く不要です。メニューから番号を選ぶだけで体験できます。高専1年生やプログラミング未経験の方でも問題なく使えます。
「WindowsによってPCが保護されました」と出た
Windows SmartScreen の警告です。「詳細情報」→「実行」を選んでください。オープンソースソフトのため、コード署名がないことが原因です。中身は全てGitHubで公開されています。
Gemini APIキーはどこで取れる？
Google AI Studio で無料で取得できます。Googleアカウントがあれば数分で発行されます。
社内のセキュリティポリシーでクラウドAPIが使えない
Ollama（ローカルAI）モードなら、データが外部に一切送信されません。初回セットアップでOllamaとモデルをダウンロードすれば、以降は完全オフラインで動作します。
研修で何人まで同時に使える？
Ollamaモード：各PCで独立動作するため人数制限なし。Geminiモード：無料枠に利用制限があるため、30名超の場合はOllamaの併用がおすすめです。
無料で使える？商用利用は？
はい、完全無料です。MITライセンスのオープンソースソフトウェアのため、商用・非商用を問わず自由にご利用いただけます。ソースコードはGitHubで公開しています。
入力したデータや会話内容は外部に送信される？
Geminiモード：入力内容はGoogle Gemini APIに送信されます。Google のプライバシーポリシーに準拠します。
Ollamaモード：全てローカルPC内で処理されるため、データは一切外部に送信されません。機密データを扱う研修ではOllamaモードをおすすめします。
Mac や Linux でも使える？
exe版はWindows専用ですが、Python版（python hajime.py）はmacOS・Linuxでも動作します。また、GitHub Codespacesを使えばブラウザだけで体験できます。
ウイルス対策ソフトにブロックされた
PyInstallerで生成したexeファイルは、一部のウイルス対策ソフトで誤検知されることがあります。ソースコードは全て公開されているため安全です。ホワイトリストへの追加、またはPython版での実行をお試しください。
